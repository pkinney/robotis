# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-12-24

### Added

- XM430 control table support for XM430 series servos (W210, W350) using Protocol 2.0
- `read_raw/3` - Read parameter values without unit conversion
- `write_raw/5` - Write raw integer values directly to registers
- GenServer accepts `:control_table` option to specify servo type

### Changed

- Control table functions now require a `control_table` module parameter
- Control tables are now behaviour modules (`Robotis.ControlTable.XL330`, `Robotis.ControlTable.XM430`)
- Model-specific functions (position resolution, baud rate mapping) moved into their respective control table modules
- Empty response case in read operations now returns `{:error, :empty_response}`

### Breaking Changes

- `ControlTable` functions no longer have default arity-1 versions; the control table module must be explicitly passed

## [0.1.1] - 2023-07-27

### Changed

- Use `cerlc` library for CRC16 computation instead of bundled binary
- Replace manual `start_link` calls with `start_supervised!/1` in tests
- Updated all dependencies

### Fixed

- Hex docs link now autogenerated correctly
- Spelling fixes
- Regenerated `.gitignore` to ignore hex.build tarballs

## [0.1.0] - Initial Release

### Added

- Dynamixel Protocol 2.0 support
- XL330-M288 control table
- Core operations: ping, read, write, sync write, fast sync read
- Factory reset, reboot, and clear commands
